# Robot Tower Defense

A tower defense game written in C++ using SFML.

## Features

- Main menu, level selection and settings
- Multiple tower types
- Enemy waves
- Bullet and explosion animations
- Inheritance and polymorphism in tower system
- GameObject base class
- Unique pointers
- Dynamic casting for tower selection
- Random enemy spawning
- Time-based movement using delta time

## Controls

- Left Mouse Button - select/build tower
- Space - start next wave
- Shift + Left Mouse Button - rocket strike
- P - pause
- Esc - back to menu
- R - restart after game over/victory
